What the project does: \
Android App for Rock Paper Scissors Game. You can play rock, paper, scissors for a great time. 

Why the project is useful: \
There's not a more balanced game than rock paper scissors. There is beauty in simplicity, especially for a balanced meta like this. This game allows a developer to create their own item for the game to use through the Item interface. Takes advantage of the observer design pattern for running the game and the strategy pattern for the AI. 

How users can get started with the project: \
Having an Android, that's about it.
